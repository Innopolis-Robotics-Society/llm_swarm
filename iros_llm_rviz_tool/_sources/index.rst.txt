iros_llm_rviz_tool
==================

RViz2 Tool plugin: click-to-send LLM goals. Operator picks a robot
    group from a popup, the click point becomes the MAPF goal, and a
    LlmCommand{mode=mapf} is dispatched on /llm/command — no LLM in the
    loop. Uses map YAML from iros_llm_orchestrator for group/bounds info.
