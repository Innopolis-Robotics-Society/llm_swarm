iros_llm_orchestrator
=====================

LLM decision action server for the Nav2 BT swarm.
    Exposes /llm/decision (iros_llm_swarm_interfaces/action/LlmDecision)
    so BT nodes can ask a Nemotron-class model for a wait / abort / replan
    verdict on the current MAPF / formation state.
