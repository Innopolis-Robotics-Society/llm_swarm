3 // Build a warm-start Solution for LNS2Solver::solve_from() from the paths
4 // of a previous plan, shifted forward in time and patched to match current
5 // robot positions.
6 //
7 // Design (v2 — fixes the "per-agent k-shift" bug):
8 // * A COMMON time shift = delta_steps is applied to ALL agents.
9 // Previously, each agent was shifted by the index at which its actual
10 // cell was found on its own prev_path ("find_actual_on_path"). That
11 // destroyed the relative timing of the schedule: a robot that fell
12 // behind got its path indexed earlier than a robot that kept pace,
13 // and the resulting seed appeared to have many "new" collisions that
14 // did not exist in the original plan.
15 // * For agents still on-plan at t=delta: the tail of the prev_path is
16 // used directly as the seed.
17 // * For agents near-but-off the expected cell: a short BFS splice is
18 // produced that joins actual_cell back onto the tail within a few
19 // steps. Falls back to a one-cell hold if the splice can't be found.
20 // * For agents that diverged far from the plan, have a changed goal,
21 // are newly introduced, or whose prev_path is no longer valid on the
22 // current map — a stub (one-cell hold) is emitted with an explicit
23 // reason.
24 //
25 // This is stateless; the caller provides everything.

72 // Per-agent classification of how the seed was produced. Useful both for
73 // diagnostics and for deciding which agents to freeze/hard-block in repair.
74 enum class SeedStatus : std::uint8_t {
75  OnTrack = 0, // actual == expected; tail of prev_path used directly
76  Spliced = 1, // actual near expected; short BFS splice prepended
77  StubFar = 2, // actual far from expected; one-cell hold ("frozen")
78  StubNew = 3, // agent was not in prev plan; one-cell hold at actual
79  StubGoal = 4, // goal changed since prev plan; one-cell hold at actual
80  StubInvalid = 5, // prev_path no longer statically valid; one-cell hold
81  StubArrived = 6, // prev plan fully elapsed and robot is at goal
82  Missing = 7, // no odometry; left empty (not in repair set)
83 };
