LlmDecision

This is a ROS action definition.

Source

# Goal: sent by MapfPlan when WARN/ERROR arrives or periodic log batch is ready
string level        # "WARN" | "ERROR" | "INFO"
string event        # human-readable description of what happened
string[] log_buffer # last N info lines accumulated before this event
---
# Result: LLM decision for BT to act on
string decision     # "wait" | "abort" | "replan"
string reasoning    # explanation shown to user (not used by BT)
---
# Feedback: LLM reasoning streamed to user while thinking
string reasoning_stream
