IROS LLM Swarm Documentation

A ROS 2 Humble research stack for orchestrating 20 differential-drive robots in a 2D Stage simulation: fleet-level MAPF planning, per-robot Nav2 navigation, leader-follower formations, a behavior-tree front-end behind a single /swarm/set_goals action, and an LLM advisory layer with three channels (reactive, proactive, operator chat) plus an RViz operator panel and click-to-command tools.

Start with Project Overview for a tour of what the project does and why, then move on to Getting Started to bring the stack up. Once the demo is running, Examples and Usage Guide walks through the operator workflow — talking to the LLM via the RViz panel, sending robots, formations, dynamic obstacles, channel triggers, dataset inspection, and the fallbacks. The System Architecture and Package Map pages are the reference for how the pieces fit together.
